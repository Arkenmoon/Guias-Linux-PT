# Guia para Fedora em Português

> Guias práticos para Fedora em Português, objectivo é ajudar utilizadores lusófonos a configurar e tirar o máximo partido do seu sistema.

![Ativo?](https://img.shields.io/badge/Ativo-Sim-brightgreen?style=flat)
![Progresso do Projeto](https://img.shields.io/badge/Progresso-40%25-yellow?style=flat)

## Sobre o Projeto

O objetivo é colmatar a falta de documentação de qualidade em PT-PT e tornar o Fedora mais acessível à comunidade portuguesa, reunindo num só lugar toda a informação relevante para uma experiência completa e funcional, especialmente quem vem do Windows.

Os guias são escritos com base em experiência real e vão sendo expandidos ao longo do tempo.

## Objetivos

- Criar documentação em PT-PT focada na distribuição Fedora
- Ajudar novos utilizadores a migrar do Windows para esta distribuição Linux
- Partilhar conhecimento com a comunidade lusófona
- Contribuir para o crescimento do Linux em Portugal
