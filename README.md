# Guia para Fedora KDE em Português

> Guias práticos para Fedora KDE em Português, objectivo é ajudar utilizadores lusófonos a configurar e tirar o máximo partido do seu sistema.

## Sobre o Projeto

O objetivo é colmatar a falta de documentação em Português de Portugal e tornar o Fedora mais acessível à comunidade portuguesa, para uma experiência completa e funcional, especialmente quem vem do Windows.

Os guias são escritos com base na minha experiência ao usar este Sistema Operativo.
