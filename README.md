# Guias para Fedora KDE em Português

> Guias práticos para Fedora KDE Plasma em Português de Portugal, para ajudar utilizadores lusófonos a configurar e tirar o máximo partido desta distribuição.

![Ativo?](https://img.shields.io/badge/Ativo-Sim-brightgreen?style=flat)
![Progresso do Projeto](https://img.shields.io/badge/Progresso-60%25-yellow?style=flat)

## Sobre o Projeto

Este repositório contém guias práticos em Português de Portugal para o Fedora KDE Plasma. 
O objetivo é colmatar a falta de documentação de qualidade em PT-PT e tornar o Fedora KDE mais acessível à comunidade portuguesa, reunindo num só lugar toda a informação relevante para uma experiência completa e funcional.

Os guias são escritos com base em experiência real e vão sendo expandidos ao longo do tempo.

## Objetivos

- Criar documentação em PT-PT focada no Fedora
- Ajudar novos utilizadores a migrar para Fedora KDE
- Partilhar conhecimento com a comunidade lusófona
- Contribuir para o crescimento do Linux em Portugal
